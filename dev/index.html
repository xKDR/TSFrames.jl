TimeFrames.jl

TimeFrames provides a convenient interface for performing standard manipulations of timeseries data. The package uses DataFrame at it's core to allow powerful data manipulation functions while being lightweight. It is inspired by zoo and xts packages from the R world.

TimeFrames wraps a familiar syntax for timeseries operations over DataFrame type, thereby, providing the user with full set of DataFrame functionalities as well. Integrations with other packages in the Julia ecosystem which are supported by DataFrames.jl come to TimeFrames at little cost.

To start using TimeFrames.jl take a look at the basic demo and then head to the User guide.
